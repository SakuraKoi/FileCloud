# SakuraKooi的外链盘
存一点奇奇怪怪的东西, 基本上是用来在代码里访问的

`mods` 发布的Mods

### HidePassword
> 隐藏 /register /login 等命令后面的密码

主要用于直播\
Forge 1.8 -> `hidepassword-1.0.0.jar` \
Forge 1.12.2 -> `hidepassword-1.0-1.12.2-SNAPSHOT.jar` 

支持命令列表:
- /login
- /l
- /register
- /reg
- .login
- .l
- .register
- .reg

> 1.12.2的版本使用了兼容性更好的实现方式, 但是有点渲染bug (能用就行
